# Magnum

My personal workspace for work-related projects, notes, and experiments.

## Overview

This repository is a central home for the various things I'm working on —
small projects, scratch code, reference notes, and one-off experiments. Rather
than scattering work across many places, everything lives here so it's easy to
find, back up, and revisit later.

## Structure

As the workspace grows, projects will be organized into their own top-level
folders, for example:

```
magnum/
├── projects/     # self-contained projects
├── notes/        # reference notes and documentation
└── experiments/  # scratch work and prototypes
```

## Getting Started

To work with this repository locally:

```bash
git clone https://github.com/joelangston0u812/magnum.git
cd magnum
```

## Contributing

This is a personal workspace, but changes are made through pull requests to keep
a clean history and make it easy to review what changed and why.
